## 🎯 About Me

I'm an **Information Security Enthusiast** with a passion for **Offensive Security (Red Team)** — but my edge comes from my deep roots in **Defensive Security (Blue Team)**.  I've done some **purple team** work along the way, which is honestly where things get interesting: watching both sides of the same attack in real time teaches you more than either side does alone.

I like to **code** — it's how I started, and it's still how I think through most problems. I'm always curious about **new technologies** and how their security models hold up (spoiler: usually not great, at first).

When I get the chance, I also enjoy **teaching IT and security** — explaining something out loud is the fastest way to find out whether you actually understand it.

Always learning. There's no finish line in this field, just the next thing worth understanding.
